# Dr. Manu V. T. — Personal Homepage

A single-page personal academic website for **Dr. Manu V. T.**, Assistant Professor in Cyber Security & Digital Forensics at NFSU Bhopal, and Programme Coordinator for the M.Sc. Digital Forensics & Information Security programme.

The site is built as static HTML/CSS/JS with an animated network-graph background, smooth scroll navigation, and interactive accordions — no build tools or backend required.
